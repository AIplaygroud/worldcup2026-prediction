# 竞彩足球 · 世界杯娱乐模拟盘规则参考

> **用途说明**：本文件为 `prediction-skill` 配套的**娱乐模拟盘**规则参考，供模拟竞猜、社区玩法、教学演示等场景使用。  
> **非真实投注指引**；与 `skill.md` 预测输出配合时，仅作彩果判定与奖金计算的规则依据。

---

## 一、彩果判定范围

足球彩票竞猜的结果均指**全场 90 分钟（含伤停补时）**的比赛结果。

- **包含**：常规时间进球与伤停补时
- **不包含**：加时赛、点球大战

> 与预测引擎一致：淘汰赛若 90 分钟战平，彩果记为**平局（1）**；加时/点球结果不计入竞彩彩果。

---

## 二、玩法简介

提供 4 个热门玩法进行投注：

| 玩法 | 说明 |
|:---|:---|
| **胜平负** | 非让球，直接按 90 分钟比分判定主胜/平/客胜 |
| **让球胜平负** | 每场比赛设置让球数，按让球后比分判定 |
| **二合一** | 将两种玩法组合在同一投注方案中（具体场次以开售为准） |
| **二选一** | 在限定选项中二选一（具体场次以开售为准） |

## 三、胜平负 & 让球胜平负

### 3.1 胜平负（非让球）

彩果即比赛的 90 分钟胜平负结果：

| 彩果代码 | 含义 |
|:---:|:---|
| **3** | 主胜 |
| **1** | 平局 |
| **0** | 客负（主队负） |

### 3.2 让球胜平负

每场比赛均设置让球，可以是：

- **主队让客队**：让球数为**负值**（如 `-1` 表示主队让 1 球）
- **客队让主队**：让球数为**正值**（如 `+1` 表示客队让主队 1 球）

**判定方法**：将让球数加到主队进球数上，再与客队进球数比较，得出让球后的胜平负彩果。

### 3.3 举例：切尔西 (-1) 埃弗顿

即切尔西（主队）让 1 球：

| 实际比分 | 让球后比分 | 彩果 | 说明 |
|:---|:---|:---:|:---|
| 1:0、2:1、3:2 | 净胜 1 球 | **1** | 赢 1 球，让球后平局 |
| 2:0、3:0、3:1 | 净胜 2 球及以上 | **3** | 让球后主胜 |
| 0:0、1:1、0:1、1:2 | 平局或输球 | **0** | 让球后主负 |

客队让球时，让球数目按同样逻辑递增（如 `+1`、`+2` …）。

## 四、奖金计算

### 4.1 基本公式（以 2 串 1 为例）

```
竞彩奖金 = 出票指数₁ × 出票指数₂（精确到分）× 2.00 元 × 倍数
```

### 4.2 计算示例

用户选择：

- **[AC 米兰 VS 尤文]** 选主胜，出票指数 **1.77**
- **[乌迪内 VS 国米]** 选主负，出票指数 **1.85**
- 过关方式：**2 串 1**

若全部猜中：

1. `1.77 × 1.85 = 3.2745`
2. 按规则精确到分 → **3.27**
3. 单倍奖金 = `3.27 × 2 = 6.54` 元

若 **100 倍**投注：

```
奖金 = 3.27（不按 3.2745 计算）× 2 × 100 = 654.00 元
```

### 4.3 其他规则

| 规则 | 说明 |
|:---|:---|
| **投注区显示金额** | 为每 1 元对应的中奖奖金 |
| **单注保底奖金** | 中奖者单注奖金不足 2 元时，补足至 2 元 |
| **指数连乘精度** | 多关过关时，各场出票指数连乘后先精确到分，再乘以 2 元与倍数 |

### 4.4 N 串 1 通用公式

```
奖金 = (出票指数₁ × 出票指数₂ × … × 出票指数ₙ)（精确到分）× 2.00 元 × 倍数
```

其中 N 为过关场次数（如 3 串 1 则 N = 3）。

## 五、名词解释

### 定胆

胆拖投注将**胆码**穿插到**拖码**中组成有效投注；每一注都含有胆码，因此选胆极为关键。

| 过关类型 | 胆错后果 |
|:---|:---|
| **N 串 1** | 胆错则整注无法中奖 |
| **组合过关** | 胆错但拖码正确时，仍可能有部分中奖 |

### 过关

一注方案竞猜多场比赛。所猜场次**必须全部正确**才能中奖。

| 名称 | 含义 |
|:---|:---|
| **2 串 1** | 猜 2 场，2 场全对才中奖 |
| **3 串 1** | 猜 3 场，3 场全对才中奖 |
| **N 串 1** | 猜 N 场，N 场全对才中奖 |

### 组合过关

从所选多场比赛中挑选部分场次组合成小串过关。

**示例**：选 5 场比赛，组合 **3 串 1** 过关

- 从 5 场中任选 3 场组成一注 3 串 1
- 共 **C(5,3) = 10** 注 3 串 1
- 好处：5 场错了 1 场或 2 场，只要某一组 3 场全对，仍可中该组 3 串 1

## 六、与预测引擎的对接提示

将 `skill.md` 的 `predictedScore` 转为模拟盘彩果时：

1. 解析比分为主队进球 : 客队进球（仅 90 分钟）
2. **胜平负**：直接比较进球数 → 3 / 1 / 0
3. **让球胜平负**：主队进球 + 让球数 与 客队进球比较 → 3 / 1 / 0
4. 淘汰赛 90 分钟平局记为 **1**；加时/点球胜负不参与竞彩彩果
